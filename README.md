# Iterators and Generators Project

Проект реализует итераторы и генераторы для работы с числовыми последовательностями
---

## Структура проекта

```
ex_9
│
├── app.py         # Запуск приложения
├── service.py     # Логика ввода и управления программой
├── iterators_generators.py   # Итераторы и генераторы
```

---

## Запуск программы

```
python app.py
```

---

## Возможности

### Fibo (итератор)

Класс `Fibo` реализует итератор для чисел Фибоначчи.

Последовательность:

```
0, 1, 1, 2, 3, 5, 8, ...
```
* Реализует методы `__iter__` и `__next__`
---

### integers (генератор)

Генератор возвращает все неотрицательные целые числа:

```
0, 1, 2, 3, 4, ...
```
* Начинается с 0
* Использует `yield`

---

### primes (генератор)

Генератор возвращает простые числа:

```
2, 3, 5, 7, 11, 13, ...
```
* Простое число — число больше 1, имеющее только два делителя: 1 и само себя
---

## Ввод данных

Пользователь вводит количество элементов, которые необходимо вывести.

Пример:

```
Сколько чисел вывести? 5
```

---

## Обработка ошибок

* Ввод не числа → сообщение об ошибке
* Ввод отрицательного или нулевого значения → сообщение об ошибке
* Программа не завершается при ошибке, а предлагает повторный ввод
